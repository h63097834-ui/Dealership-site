Description
This is a bike dealership app written in the MERN stack primary tools and languages used include 
1) Java Script
2) Express
3) React
4) Mongo DB
5) and some other tools such as jwt , bcrypt etc
Please not this is supposed to serve as a portfolio app and not supposed to be a production ready app.

More About It
As u know the stack , tools and languages used a little about the app structure and use is.
It allows a user to create their account Log In to existing accounts see account details.
Purchase a product it has limiting as a user cannot purchase more then 3 orders at a time.
A user can logout and delete their account an account cannot be deleted if their are orders pending.
It also has admin features such as adding a new bike product or remove a bike product.
It utelizes cloudinary integration for handling bike images.
it also provides 4 brand pages.
Users can also filter bikes based on company names e.g Yamaha , Honda etc.
They can also find a specific bike by company name and bike name e.g Yamaha R3 etc.
